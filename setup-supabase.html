Setup Supabase untuk PT PMP

Step 1: Buat Table & Storage di Supabase SQL Editor

Buka SQL Editor di Supabase Dashboard, lalu copy-paste dan jalankan SQL ini:

-- Create photos table CREATE TABLE IF NOT EXISTS public.photos ( id TEXT PRIMARY KEY, album TEXT NOT NULL, caption_id TEXT DEFAULT '', caption_en TEXT DEFAULT '', src TEXT NOT NULL, "order" INTEGER DEFAULT 0, type TEXT DEFAULT NULL, created_at TIMESTAMPTZ DEFAULT now(), updated_at TIMESTAMPTZ DEFAULT now() ); CREATE INDEX IF NOT EXISTS idx_photos_album ON public.photos (album); -- Enable RLS ALTER TABLE public.photos ENABLE ROW LEVEL SECURITY; -- Everyone can read CREATE POLICY "Public read" ON public.photos FOR SELECT USING (true); -- Everyone can insert CREATE POLICY "Public insert" ON public.photos FOR INSERT WITH CHECK (true); -- Everyone can update CREATE POLICY "Public update" ON public.photos FOR UPDATE USING (true); -- Everyone can delete CREATE POLICY "Public delete" ON public.photos FOR DELETE USING (true); -- Create storage bucket INSERT INTO storage.buckets (id, name, public) VALUES ('gallery', 'gallery', true) ON CONFLICT (id) DO NOTHING; -- Storage: public read CREATE POLICY "Public read gallery" ON storage.objects FOR SELECT USING (bucket_id = 'gallery'); -- Storage: public upload CREATE POLICY "Public upload gallery" ON storage.objects FOR INSERT WITH CHECK (bucket_id = 'gallery'); -- Storage: public update CREATE POLICY "Public update gallery" ON storage.objects FOR UPDATE USING (bucket_id = 'gallery'); -- Storage: public delete CREATE POLICY "Public delete gallery" ON storage.objects FOR DELETE USING (bucket_id = 'gallery');

Step 2: Verifikasi & Seed Data

Setelah SQL di atas berhasil dijalankan, klik tombol di bawah untuk verifikasi dan seed data awal.
